# Fonts

## solarized dejavu

[![dejavu-16](https://github.com/bewest/homeware/raw/master/fonts/font-dejavu-16.png)](#solarized dejavu)\

```
WM_CLASS(STRING) = "xterm", "SolarBlue"
WM_COMMAND(STRING) = { "/usr/bin/xterm", "-class", "SolarBlue", "-fa", "dejavu-16" }
```
